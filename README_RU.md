# 🧠 Система определения фейковых новостей

Полноценная ML-система для классификации новостей с использованием нейронных сетей, ансамблей моделей и развёртыванием в виде API.

---

## Обзор проекта

В рамках проекта реализован полный цикл разработки ML-системы:

- Предобработка и очистка текстовых данных
- Векторизация с использованием Word2Vec
- Обучение моделей: RNN, LSTM, GRU
- Автоматический подбор гиперпараметров (Optuna)
- Построение ансамбля моделей
- Развёртывание через FastAPI
- Контейнеризация с использованием Docker

---

## Архитектура

Данные → Предобработка → Токенизация → Word2Vec → Модели → Ансамбль → API → Docker

---

## Предобработка данных

- Объединены признаки:
  - `title`
  - `news_url`
  - `source_domain`
- Из URL удалены доменные части с сохранением смысловых токенов
- Применены:
  - приведение к нижнему регистру
  - очистка регулярными выражениями
  - удаление стоп-слов
- Сформирован итоговый признак `text`

---

## Векторизация текста

Использован **Word2Vec** вместо классических подходов (TF-IDF / Bag-of-Words):

- учитывает семантические связи между словами
- улучшает обобщающую способность моделей
- интегрирован напрямую в embedding слой PyTorch

---

## Модели

Реализованы и сравнены три архитектуры:

- **RNN** — базовая модель
- **LSTM** — учитывает долгосрочные зависимости
- **GRU** — оптимальный баланс скорости и качества

---

##  Подбор гиперпараметров

Использована библиотека **Optuna** для автоматической оптимизации:

Подбирались:
- размер скрытого слоя
- количество слоёв
- dropout
- learning rate

Основная метрика:

ROC AUC

---

## Ансамбль моделей

Построен ансамбль из:

RNN + LSTM + GRU

- используется взвешенное усреднение вероятностей
- веса моделей оптимизированы с помощью Optuna
- повышена стабильность и качество предсказаний

---

## Метрики

Для оценки моделей использовались:

- Accuracy
- F1-score
- ROC AUC (ключевая метрика)

---

## API (FastAPI)

Реализован REST API:

### Endpoint

POST /predict

### Пример запроса

{
  "text": "Breaking news: government announces new policy"
}

### Пример ответа

{
  "model_probabilities": {
    "RNN": 0.58,
    "LSTM": 0.21,
    "GRU": 0.00
  },
  "ensemble_probability": 0.18,
  "ensemble_prediction": 0,
  "label": "fake"
}
